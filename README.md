# faceless-tiles

Oeffentlich gehostete PNG-Kacheln fuer die faceless KI-Content-Engine.
Wird automatisch von faceless_engine/publish/hosting.py befuellt.

Dateien liegen unter `<wochen-datum>/tiles/day<N>_slide<M>.png` und sind ueber
`https://raw.githubusercontent.com/kesa2404-collab/faceless-tiles/main/<pfad>` erreichbar.
